## 移动端使用 console 插件

实例

```html
<!-- <!DOCTYPE html> -->
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width, viewport-fit=cover" />
    <title>Your App</title>

    <script src="https://cdn.bootcss.com/vConsole/3.2.0/vconsole.min.js"></script>
    <script>
      var vConsole = new VConsole();
    </script>
  </head>
  <body>
    <div id="root"></div>
  </body>
</html>
```
