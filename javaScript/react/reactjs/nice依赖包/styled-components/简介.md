# styled-components 

- styled-components 是针对React写的一套css in js框架，简单讲就是在js中写css.  
  相对于预处理几器（sass,less）, 语法更简单。   


## 基础用法

```js
  import styled from 'styled-components';

  const HomeWrapper = styled.div `
    width: 960px;
    margin: 0 auto;
    overflow: hidden;
  `;

  const HomeLeft = styled.div `
    float: left;
    width: 625px;
    margin-left: 15px;
    padding-top: 30px;
    .bannder-img {
      width: 625px;
      height: 270px;
    }
  `;
  const HomeRight = styled.div `
    float: right;
    width: 280px;
    margin-left: 15px;
    padding-top: 30px;
  `;

  render () {
      return (
          <HomeWrapper>
              <HomeLeft>
                  left
              </HomeLeft>
              <HomeRight>
                  right
              </HomeRight>
          </HomeWrapper>
      )
  }
```
