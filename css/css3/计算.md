## calc

css中 长度值计算函数  

```
	div {
		width: calc(100% - 100px); // div父元素宽度 - 100px 
	}
```

## counters

对部分或子部分编号

```
	<ul>
    <li>List item</li>
    <li>List item</li>
    <li>List item
      <ul>
        <li>list item</li>
        <li>list item</li>
        <li>list item</li>
      </ul>
    </li>
  </ul>

 ... 


 ul {
	  counter-reset: section;
	}

	li::before {
	  counter-increment: section;
	  content: counters(section,'.') ' ';
	}


```
